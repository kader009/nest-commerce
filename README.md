# Assignment Submission: Nest Mart Landing Page

A fully responsive eCommerce landing page built for the Frontend Web Development assignment.

## Technical Stack

- HTML5 & Vanilla CSS3
- Flexbox & Grid Layouts
- Responsive Breakpoint: **480px** (Mobile Optimized)
- Tooling: Vite

## Repository Structure

```text
/
├── index.html       # Entry Point
├── src/             # Styles & JS
├── public/image/    # Assets
└── package.json     # Vite Config
```

## How to Run

1. **Clone**: `git clone https://github.com/kader009/nest-commerce.git`
2. **Setup**: `npm install`
3. **Run**: `npm run dev`
